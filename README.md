### Installation

Wizard site requires [Python](https://www.python.org/) v3+ to run. 

Install the dependencies and devDependencies and start the server.

```sh
...\> pip install Django

...\> py manage.py migrate
...\> py manage.py makemigrations posts

...\> py manage.py runserver
```
